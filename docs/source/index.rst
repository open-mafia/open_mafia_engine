.. Open Mafia Engine documentation master file

Welcome to Open Mafia Engine's documentation!
=============================================

This Python library, :mod:`mafia`, is currently under 
(semi-)active development. The aim is to create a consistent, 
extendable engine for Mafia/Werewolf-like games.

The Git repository can be found here (dev branch): 
https://github.com/open-mafia/open_mafia_engine/tree/dev

`Stable documentation (likely to be outdated at this point)
<https://github.com/open-mafia/open_mafia_engine/>`_
